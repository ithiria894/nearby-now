# 市場證據 — Threads 開團 post

呢個 folder 係 enoki 嘅 target-user / 市場 gap 證據：一批真實網民喺 Threads 公開出「臨時開團 / 有冇人一齊玩」嘅 post（Nicole 2026-07-16 蒐集）。呢個 index map 返每張圖對應嘅痛點。

## 核心 thesis

每個城市都有一種人：**想搵人一齊做活動、但唔想做「host」嗰個好認真、好有責任嘅角色**。
佢哋而家靠 Threads/IG 出 post → 留言「報名/+1」→ 樓主手動 DM + IG form 數人頭 + 拉 WhatsApp group。
**痛點唔止「冇 RSVP 工具」，係「主辦/host 角色太重」。** enoki 要拆走 host 負擔 = 架構層 differentiator。

## enoki 點解呢啲

- **附近有人開咗活動 push**（已 build）→ 殺「discovery 靠彩數,post 被淹沒」
- **匿名輕身份**（nickname+性別、隨時改）→ 對應「唔想又 upload 頭像又寫興趣搞到似交友 app」
- **內建 RSVP + 容量** → 取代 IG form / 手動 DM queue
- **Rooms tab** → 開團後 coordination home，取代 WhatsApp group
- **拆走 host 負擔** → 發起人 = 「先開口嘅參加者」，唔係要孭場嘅主辦（⚠️ 留意 CY 加咗 host 金冠，唔好令 host 變成卻步嘅認真身份）
