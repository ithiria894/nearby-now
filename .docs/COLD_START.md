# enoki — 冷啟動策略 / Cold-Start Playbook ❄️🍄

> **冷啟動係 enoki 嘅 #1 生死線**(見 `MARKET_RESEARCH.md`:Highlight 就係咁死,連同市場、有錢嘅 Flock 都仲未 crack)。
> 「附近邊個而家想做 X」呢類 app,冇本地密度就係死城 → 冇人返 → 死。呢份 doc = 點樣唔死。

---

## 0. 你有一張別人冇嘅底牌:web 分享 link

**web share-link(VISION §5)唔止係漏斗,係冷啟動嘅破解法。**

因為 organizer 掉條 link 落佢**現有嘅 WhatsApp / Threads / IG group**(嗰度已經有人),enoki **由 user #1 就有用 —— 唔使自己先養一大堆人**。即係你**借人哋嘅 liquidity**,唔係由零起自己嗰堆。

> 呢個係 two-sided product 打贏冷啟動嘅經典解:**single-player mode 先有價值**(一個 organizer + 佢現有 group 就 work),再滾網絡效應。
> Flock 係 native-app-first,好可能做唔到呢招 —— 呢個可能就係你贏佢嘅位。

---

## 1. 打法(順序執行)

### ① 寄生現有 group,唔好自己養人先 🎯

Threads 啲「火鍋局 / 找搭子 / 開團」organizer **已經有 crowd**。畀 enoki 佢哋做**更好嘅工具**:照樣 Threads 出 post,但「報名」link = 一間 enoki 房(內建 RSVP + 容量 + chat,取代佢手動 DM + Google form + WhatsApp group)。
→ enoki 騎住 Threads 嘅流量;organizer 班 follower 一次 event 變一批 enoki user。**enoki 由 user #1 就解決緊真痛點。**

### ② 窄 + 密:揀一個 wedge 社群,飽和佢

- 最明顯 wedge = **溫哥華華人 / 國際生**(開團行為、市場證據、你、Flock 全部喺度),或者一間 campus(UBC / SFU)。
- **密度 > 覆蓋。** 唔好一開始就「全世界」。
- 招 10–20 個「常開團」嘅人做 supply 種子(the people already doing 火鍋 / 唱K / board game)。**人手 seed,do things that don't scale。**

### ③ 喺時間上集中(波次),唔淨係空間

稀疏用戶攤喺「anytime」= 永遠空。集中做「**逢週五溫哥華**」主題夜 push → 大家同一時間出現 → 就算總數少都有**時間密度**。
_(Timeleft 週會、No More Lonely Friends 逢週六 都係呢招。)_

### ④ 永遠唔好睇落死城

- feed show **過去 + 更遠半徑**嘅活動(FOMO,VISION §9)—— 新用戶開 app 唔會見空白。
- **broadcast push(你已 build)** 令得嗰幾個活動 reach 到附近所有人 → 把稀疏供應**聚焦成可見需求**。

### ⑤ 創辦人親自 seed 供應

你 + 早期用戶親自開真活動 + 出席,做「先開口嘅參加者」,撐到有機供應接手。
_(Reddit、Meetup、每個 marketplace 都係咁起步 —— 唔可恥,係必須。)_

### ⑥ 餐廳 / 場地合作 = 現成供應(VISION §6)

「韓燒 happy hour 要最少 2 人」= 一個現成活動理由。喺 wedge 度 seed 呢啲,一石二鳥(供應 + revenue hook)。

---

## 2. 心法(一句)

> 你**唔使全球贏冷啟動**。你只要:
> **①** 對**一個 organizer + 佢現有 group**,用條 link 就有用(single-player);
> **②** 喺**一個密 wedge**,用時間 + 空間飽和;
> 然後**複製去下一個 wedge**。
> **條 share-link 係關鍵 —— 借人哋嘅人流,唔使由零起。**

---

## 3. 點知 work 緊(早期指標,唔好睇裝機量)

- **每個 wedge 嘅「開房 → 有人 join」成功率**(唔係下載量)—— 開咗房冇人 join = 冷啟動未過。
- **同一 wedge 內嘅 D7 / D30 retention**(飽和一個先睇,唔好被廣覆蓋嘅平均數呃)。
- **organic 供應比例**:幾多 % 活動係非-種子用戶開嘅(種子退場後仲有冇人開房)。
- **link → 入房 → 留低下載** 轉換率(§0 個 cheat code work 唔 work 嘅命脈)。

⚠️ **反指標**:全城/全國「總用戶數」升但**每個 wedge 密度唔夠** = 攤薄咗、假成長(IRL / Highlight 死法)。寧願一個 wedge 好過十個城市都係死城。
